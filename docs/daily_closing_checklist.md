# Daily Closing Checklist

## 1. Before closing

1. Finish all active bills.
2. Ensure pending cart is empty.
3. Enter all purchases and expenses for the day.

## 2. Validate reports

1. Open Reports > Overview and set Today filter.
2. Confirm Sales, COGS, Expenses, and Realistic Daily Profit.
3. Open Reports > Stock and Audit and review low-stock list for next-day planning.

## 3. Close-day action

1. Run Close Day (Ctrl+L) from Reports > Data Ops.
2. Verify closure summary popup.

## 4. Data safety

1. Go to Reports > Data Ops and click Backup Now.
2. Confirm backup file created successfully.
3. Ensure scheduled backup is enabled with appropriate interval.

## 5. Optional exports

1. In Reports > Data Ops, run Export All CSV for accountant handoff.
2. In Reports > Data Ops, run Export XLSX and Printable Summary for records.

## 6. End of day sign-off

1. Check Reports > Stock and Audit > Audit Log for unusual sensitive actions.
2. Exit app cleanly.
